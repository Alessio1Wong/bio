Remote Chemistry Lab Assistant

Project Overview

Designed a remote chemistry lab assistant to facilitate junior high school chemistry experiments such as dissolution, filtration, and evaporation. The assistant addresses the challenges of inaccessible laboratory resources, using an automated box controlled via a network for remote access.

This project won the First Prize in the Shanghai Youth Science and Technology Innovation Competition and participated in China International College Students' Innovation Competition (2023) - Emerging Track.

Features

  • Multi-functional areas for chemicals, instruments, and waste handling
  • Automated robotic arm for precise handling of equipment
  • Fire safety measures with emergency shut-off systems
  • Real-time monitoring using ESP32CAM with interactive controls

Innovation

By integrating ESP32CAM and Arduino, the assistant supports online and offline hybrid education, promoting equal access to laboratory resources while preparing for future educational innovations.
